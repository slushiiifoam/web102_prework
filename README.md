# WEB102 Prework - *Sea Monster Crowdfunding*

Submitted by: **Sebastian Graciano Munoz**

**Sea Monster Crowdfunding** is a website for the company Sea Monster Crowdfunding that displays information about the games they have funded.

Time spent: **5** hours spent in total

## Required Features

The following **required** functionality is completed:

* [x] The introduction section explains the background of the company and how many games remain unfunded.
* [x] The Stats section includes information about the total contributions and dollars raised as well as the top two most funded games.
* [x] The Our Games section initially displays all games funded by Sea Monster Crowdfunding
* [x] The Our Games section has three buttons that allow the user to display only unfunded games, only funded games, or all games.

The following **optional** features are implemented:

* [x] List anything else that you can get done to improve the app functionality!
* [x] Interactive Search Bar: Implemented a real-time search feature using an input event listener and the .filter() method to allow users to find specific games by name instantly.

* [x] Navigation Bar: Created a sticky header containing navigation links that allow users to jump to the About, Stats, and Games sections smoothly.

* [x] Responsive Mobile Design: Added CSS Media Queries to ensure the layout remains functional on smaller screens, automatically switching from multi-column rows to single-column vertical stacks.

* [x] Enhanced Styling: Improved readability by adding rounded, semi-transparent background boxes to content sections. Upgraded game cards with hover effects (lift and shadow) to provide visual feedback. Polished button aesthetics and interactions.
